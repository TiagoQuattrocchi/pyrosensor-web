Pyrosensor

Este proyecto se basa en el sensado de temperatura mediante una termocupla. En base a un valor elegido mediante un selector, se mantendrá en la temperatura deseada mediante un control proporcional de una llave de paso de gas. Está diseñado para uso industrial.
